Two clicks to a live AI2FI dashboard. No install, no Terminal.

Two clicks to start

You're already in Finder — that's where you found this page. Switch back to that window. You'll see Start-Demo.command sitting next to this file.

1

Right-click Start-Demo.command → click Open

Right-click (or Control-click) the file. Choose Open from the menu.

Why right-click? macOS blocks downloaded scripts from running on a plain double-click the first time. Right-click → Open is how you tell it "yes, I trust this." After the first time, double-click works.

2

Click Open in the security dialog

macOS will ask "Are you sure you want to open it?" — click Open.

A Terminal window opens with the demo running, and your browser opens the dashboard at http://localhost:5173 after a few seconds.

To stop the demo: close that Terminal window. That's it.
