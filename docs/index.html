AI2FI

A local-first financial coaching platform. Claude is the coach. A small web dashboard, running on your machine, is the window into your own data. Nothing is transmitted, no account to create.

1. Download the repo

AI2FI is a plain git repository. You can either clone it or download a ZIP — both work, both leave everything on your machine.

Option A — Download ZIP (simplest)

Download AI2FI.zip

Unzip it anywhere you like — Documents, Desktop, a projects folder. The folder name after unzipping will be AI2FI-main; you can rename it to AI2FI if you prefer.

Option B — Git clone (if you have git)

git clone https://github.com/anorby515/AI2FI.git
cd AI2FI
macOS only for now. The one-click installer (setup.command) is macOS-specific because it uses launchd to auto-start the dashboard at login. Linux and Windows users can still run the dashboard manually — see dashboard/README.md for the non-scripted path.

2. Set up the dashboard

The dashboard reads a spreadsheet on your machine and renders portfolio, net worth, and benchmark views. It runs at http://localhost:3001 and auto-starts when you log in.

One-time install (macOS)

  1. In Finder, open the AI2FI folder you just downloaded.
  2. Open the dashboard/ subfolder.
  3. Double-click setup.command.
  4. A Terminal window opens and walks through: Homebrew check → Node install if needed → npm install → build → register login agent → open the dashboard in your browser.
First-run Gatekeeper hurdle. macOS may refuse to run setup.command the first time because it was downloaded from the internet. If that happens: right-click setup.command → Open → Open in the dialog. It runs once with your permission and macOS remembers afterward.

To uninstall

Double-click uninstall.command in the same dashboard/ folder. It unloads the login agent and removes the plist. Your project files, spreadsheet, and user profile are untouched.

Non-Mac (or "I want to run it manually")

You'll need Node 18+ installed. From the repo root:

cd dashboard
npm install
cd client && npm install && npm run build && cd ..
npm start

Then open http://localhost:3001. Full details and the dev workflow are in dashboard/README.md.

3. Start coaching with Claude

The dashboard is the "seeing" half. Claude is the "thinking" half — and that's where the work actually happens. The entry point is a single skill.

Open Claude in this folder

Open your preferred Claude interface (Claude Code, Claude Desktop, or claude.ai/code) with the AI2FI directory as the working folder. Claude will automatically pick up the skills in skills/ and the curriculum in core/ and modules/.

Run the entry skill

In Claude, type:

/financial-check-in

That's the front door. It runs the master assessment, consults the financial order-of-operations, and routes you into whichever module matches where you are today. First-time users get the full two-pass assessment; returning users get the lightweight "what's changed since last time?" version.

Other skills you can call directly

  • /module-coach <module-name> — dives deep into one topic (e.g. /module-coach investing).
  • /knowledge-check <topic> <level> — standalone quiz, tiered fundamentals → intermediate → advanced.
Your data stays yours. Personal financial files live under user-profiles/<you>/private/ and are gitignored. Session notes, goals, and competency levels are written by Claude at session close into user-profiles/<you>/ — also gitignored. Nothing under user-profiles/ is ever committed except the example/ template.
