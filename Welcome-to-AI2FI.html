1. Install the dashboard

You've downloaded AI2FI and opened this file. One more step gets the dashboard running at http://localhost:3001 and auto-starting at login.

Open Terminal (⌘-Space, type "Terminal") and paste the command below. It's pre-filled with the full path to your AI2FI folder:

Install command — copy and paste into Terminal
cd "<detecting…>" && bash dashboard/setup.command

Folder detected as: (detecting…)

The installer will:

  1. Check for Homebrew and Node, install them via Homebrew if missing.
  2. Install dashboard dependencies and build the client.
  3. Register a macOS login agent so the dashboard auto-starts.
  4. Open http://localhost:3001 in your browser.

First run takes a couple of minutes. You'll be prompted for your password once so the installer can register the login agent.

Why Terminal and not double-click?

Double-clicking dashboard/setup.command from Finder triggers macOS Gatekeeper because the file was downloaded from the internet — it blocks with a "cannot be opened" dialog that only offers "Move to Trash" or "Done". The Terminal path above reads the script with bash, which doesn't invoke the Gatekeeper check. Same script, same install, just no ceremony.

If you've already cloned this repo with git (not downloaded a ZIP), double-clicking setup.command works normally — git-cloned files aren't quarantined.

Non-Mac / manual install

The installer script uses launchd (macOS only), but the dashboard server itself runs anywhere Node does. You'll need Node 18+. From Terminal:

cd dashboard
npm install
cd client && npm install && npm run build && cd ..
npm start

Then open http://localhost:3001.

2. After install

When the installer finishes, your browser opens http://localhost:3001 and you're looking at the dashboard. It'll be mostly empty until you drop in your spreadsheet.

Drop in your spreadsheet

The dashboard reads a local spreadsheet to render portfolio, net worth, and benchmark views. Put yours at:

user-profiles/<your-name>/private/Finances.xlsx

Start from user-profiles/example/ as a template if you want the expected shape. Everything under user-profiles/<your-name>/ is gitignored — it never leaves your machine.

Uninstall anytime

From Terminal, in this folder:

bash dashboard/uninstall.command

Unloads the login agent, removes the plist, kills anything on port 3001. Your files, spreadsheet, and profile are untouched.

3. Start coaching with Claude

The dashboard is the "seeing" half. Claude is the "thinking" half — and that's where the work happens.

Open Claude in this folder

Open Claude Code, Claude Desktop, or claude.ai/code and point it at this AI2FI folder as the working directory. Claude automatically picks up the skills in skills/ and the curriculum in core/ and modules/.

Run the entry skill

In Claude, type:

/financial-check-in

That's the front door. It runs the master assessment, consults the financial order-of-operations, and routes you into whichever module matches where you are today. First-time users get the full two-pass assessment; returning users get the lightweight check-in.

Other skills

  • /module-coach <module-name> — dives into one topic (e.g. /module-coach investing).
  • /knowledge-check <topic> <level> — standalone quiz, tiered fundamentals → intermediate → advanced.
Your data stays yours. Personal financial files live under user-profiles/<you>/private/ and are gitignored. Session notes, goals, and competency levels are written by Claude at session close — also gitignored.
