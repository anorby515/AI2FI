# Financial Strategy — Module Metadata

**Role in the system:** This is the north-star module. Every other module feeds into it, and users return here every quarter to set goals, revise them, and take stock.

**Structure:** The fundamentals knowledge check is a three-part conversation rather than a standalone quiz.

1. **Part 1 — Getting to know each other.** Conversational assessment that covertly maps comfort and competency across the 14-step Financial Order of Operations.
2. **Part 2 — The Order of Operations.** Walkthrough of the framework with pause-points for questions.
3. **Part 3 — Goal setting.** One conversation per FOO step: do you have a goal here? Are you ready for one? If not, what comes first?

**Output:** A populated `user-profiles/[userid]/goals.md` organized by FOO step, and a journal entry capturing comfort levels and notable decisions.

**Prerequisites:** None. This is usually the user's first session.

**Revisit cadence:** Quarterly, triggered by `/financial-check-in`.
